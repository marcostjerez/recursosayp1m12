01 - Tipos de Datos

01 - Tipos de Datos

Ya habíamos hablados que para el diseño de algoritmos existen cuatro tipos de datos que son el Entero, Real, Lógico y Carácter, en Python existen tres numéricos int (entero), float (reales) y complex (complejos), pero no son definidos gracias a las características de tipado dinamico, aunque si un valor tiene un tipo puedes reasignarlo a otro tipo como vamos a poder ver en la práctica, en C/C++ existen el extensor double, que es un float con el doble de precisión, el void que es una variable sin valor y también conseguiremos los modificadores long, short, unsigne y signe. En Python estos modificadores y extensores existen de manera estructural, pero no hace falta definirlos, también gracias al tipado dinámico, por ejemplo, en las versiones por debajo del Python 3 debías declararlo como long o como int dependiendo del tamaño del valor, pero de la versión 3 en adelante solo debes darle el valor, el intérprete lo asume como int por defecto y el define el rango según su amplitud.
